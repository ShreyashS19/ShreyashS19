<p>
  <img src="https://img.shields.io/badge/PIN%20Validation-critical?style=flat-square" />
  <img src="https://img.shields.io/badge/Atomic%20Transactions-success?style=flat-square" />
  <img src="https://img.shields.io/badge/Audit%20Logs-blue?style=flat-square" />
  <img src="https://img.shields.io/badge/Excel%20Reports-informational?style=flat-square" />
  <img src="https://img.shields.io/badge/Email%20Notifications-Gmail-D14836?style=flat-square&logo=gmail&logoColor=white" />

</p>


**Role:** Full-Stack Developer  
**Summary:** : A full-stack banking activity simulation platform that models real-world banking workflows with secure transactions, audit trails, Excel reports, and real-time Gmail email notifications.

**Key points**
- Secure transfers: PIN & account-status validation plus sufficient-balance checks.  
- Atomic updates: Balance debits/credits applied together to avoid inconsistencies.  
- Observability: Transaction logs, email confirmations and Excel export for reporting.  
- Safe-by-design: Validators and layered architecture to enforce business rules.  
**Tech:** Java (Jersey / Spring-ready), MySQL, React (frontend for demos), Docker  
**Repo:** https://github.com/ShreyashS19/Bank-Simulator-Intern.git
